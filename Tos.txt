Data logging
The Discord bot logs a variety of activities on a daily. The bot will only log activity that relates to this specific discord bot it will not log any messages sent, modifications to the server, or anything else that does not relate to the Discord bot. The bot will log almost all commands that are used especially ones that upload data to the database.

Server Data
All data that is uploaded to the bot is stored in a database. The database only stores data that relates to the bot, examples would be fruit tables, custom color, or restrict mode. The bot will also store the server’s ID as that is used to uniquely identify your data. All the data that you upload can be privately seen by the bot owner. Your server’s data could also be deleted if the bot was removed from the server for a long period of time.
